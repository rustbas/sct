# Changelog

## 0.2.0

- Инкрементальный `sync` (по умолчанию) и `sync --full`
- Проверка строки перед `done` / `reopen` (`line_hash`, текст, маркер)
- Коды выхода для скриптов; `sct doctor`, `sct init`
- Поиск корня проекта вверх по дереву (`.sct/`)
- `requirements.txt` для установки зависимостей
- CI: GitHub Actions + интеграционные тесты
- `list --priority`, `list --json` с полем `version`
- TUI: предупреждение при устаревшем кэше

## 0.1.0

- Первый рабочий прототип: сканер, кэш, CLI, TUI
